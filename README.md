# lab-os — Jean Moncayo's dev home

My primary agentic development home: a fork of the CAMELS Research Group
**lab-os** handbook, run in fork-as-home mode. The lab rules live natively at
`.claude/rules/`; `git pull upstream main` pulls handbook updates from upstream.

Project repos nest inside as their own gitignored repos. Plans (`_plans/`), the
project log (`project_log.md`), and the rules are tracked here at the fork root.

Published handbook (read in browser): https://camels-research-group.github.io/lab-os/
