# Onboarding Project

> **This page is superseded.** The two-week sandbox has been redirected into the
> [Workshop Program](/docs/workshops/) — a three-part sequence (Planning → Building → Closeout)
> that carries one real project through spec, execution, and retrospective. Start there.

The workshop program absorbs the goals of this sandbox: spec-driven workflow practice, autonomous
delegation, and a carry-forward artifact. It runs on your own project rather than a fixed sample,
which means the learning transfers directly.

**[Go to the Workshop Program →](/docs/workshops/)**
