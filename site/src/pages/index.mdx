<h1 style={{textAlign: 'center'}}>Spec-driven, agent-assisted development, the CAMELS way</h1>

This is the CAMELS Research Group's field guide to building with AI agents. You
set up an agentic workspace - Claude Code (Anthropic's AI coding assistant),
persistent context files, and the shared lab rules that load into every session -
then practice spec-driven development (writing down what to build, and the plan
to build it, before any code) on a low-stakes sandbox project: spec, plan,
subagent build (implementation delegated to helper AI agents), review.
