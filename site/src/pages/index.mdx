# Learn spec-driven development with an agentic workspace

This handbook is a hands-on skills-building exercise. You set up an agentic
workspace — Claude Code (Anthropic's AI coding assistant), persistent context
files, and rules that load into every session — then practice spec-driven
development (writing down what to build, and the plan to build it, before any
code) on a sandbox project: spec, plan, subagent build (implementation
delegated to helper AI agents), review.

## What you end up with

- A **working agentic workspace**: Claude Code wired to a personal `CLAUDE.md`,
  a workspace map, and a linked rule set so every session starts from the same
  conventions
- The **spec-driven development workflow** — spec → plan → subagent build →
  review — practiced end to end on a low-stakes sandbox project
- A **small dashboard-style app you built yourself** — the exercise's learning
  vehicle: every skill above gets practiced by building it, in your own
  disposable repo
